# Copilot Playbook

이 폴더는 GitHub Copilot 에이전트가 tattoo_frontend 모노레포에서 일할 때 참고해야 할 빠른 지침을 정리한 공간입니다. 핵심 섹션은 다음과 같습니다.

- `architecture/`: 패키지 경계, barrel 규칙, 레이아웃 설계 철학 등 구조적 의사결정 기록
- `workflows/`: melos 스크립트, 릴리즈 전 체크리스트처럼 자주 실행하는 절차 요약
- `ui_guidelines/`: Flutter Web 1280px 고정 캔버스 레이아웃 등 UI/UX 관련 결정 사항
- `troubleshooting/`: 실무에서 반복되는 문제와 빠른 해결 팁

새로운 문서가 필요하면 적절한 하위 폴더에 Markdown 파일을 추가하고, 이 README 의 섹션 설명도 함께 업데이트하세요.
